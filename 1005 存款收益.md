# [stackoverflow] 1005 存款收益
## 题目描述  
按年利率R%存入本钱X，存入P年后的本利合计为Y=X*((100+R)/100)^P，求Y的值。       
输入：输入一行三个空格隔开的实数R,X,P        
输出：输出收益Y，答案保留两位小数。    
样例输入：1 100 5  
样例输出：105.10  
<hr>  

## 解题思路    
这道题也很简单，计算的式子都给出来，只要注意乘方的优先级比乘法高就可以了  
  
<hr>  

## 代码      

```c
//C
#include <stdio.h>
#include <math.h>
int main(void){
  double r,x,p,y;
  scanf("%lf %lf %lf",&r,&x,&p);
  y = x * pow((100 + r) / 100,p);
  printf("%.2f",y);
  return 0;
};



```

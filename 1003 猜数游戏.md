# [stackoverflow] 1003 猜数游戏
## 题目描述  
有一个“就是它”的猜数游戏，步骤如下：请你对任意输入的一个三位数x，在这三位数后重复一遍，得到一个六位数，467-->467467.把这个数连续除以7、11、13，输出最后的商。  
输入：输入一个三位数x。  
输出：输出最后的商。  
样例输入：100  
样例输出：100
<hr>  

## 解题思路    
把一个三位数在后面重复一遍其实就等于*1001  
以467为例，467467 = 467 * 1001  
然而1001 = 7 * 11 * 13  
乘了1001后再除以7,11,13等于不变  
输入什么就输出什么就好了  
（数学方法真好用）  
  
<hr>  

## 代码      

```c
//C
#include <stdio.h>
int main(void){
	int n;
	scanf("%d",&n);
	// 1001 = 7 * 11 * 13;
	printf("%d",n);
	return 0;
};


```
